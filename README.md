SELESAI COMMIT GITHUB
Pergi ke terminal vscode di dalam project, ketik.

npm install

===============================================================================================================================================================================================================================

1. Bekerja di branch role nya masing-masing. cari, klik "Current Branch" di github desktop. pilih role tugas kalian. (Jika database klik tulisan Database)
2. Khusus untuk Frontend dan Backend buat sub branch baru dengan nama kalian.
3. setelah selesai mengupdate, pastikan kolom desktipsi di isi dengan detail.
4. jangan lupa laporan kepada ketua apa yang baru saja kalian kerjakan agar ketua bisa cek apakah sudah masuk atau belum.
